### Learnings:
- Chinese curly quotes (", ", ', ') in TypeScript source files cause SWC parse errors. Must use straight quotes only, and escape apostrophes in single-quoted strings.
- Tailwind v4 uses @theme directive for theme extension instead of tailwind.config.ts extend
- Template literal types in TypeScript are strict — dynamic key access to i18n needs string type or a Record cast
- Duplicate properties in inline style objects cause TypeScript "multiple properties with same name" errors
- Phone simulator CSS must be scoped (`.phone-scope` + `@layer components`) to avoid conflicts with landing page Tailwind utilities
